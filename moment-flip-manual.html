01 · APP OVERVIEW

이 앱이 뭘 하는 앱인가요?

한 줄로 말하면: 커플이나 가족끼리 사진 한 장으로 하루를 교환하는 프라이빗 일기장이에요.

💡

이런 분들을 위해 만들었어요

카카오톡은 너무 즉각적이고 답장 부담이 있고, 인스타그램은 너무 공개적이에요. 그 중간 어딘가 — 딱 두 사람만 보는 작은 일기장이 필요했어요.

사진 1장, 감정 아이콘 1개, 60자 이내 글귀. 하루 1분이면 충분해요.

🔑

핵심 특징 3가지

① 1:1 프라이빗 — 나와 파트너만 볼 수 있는 우리만의 공간
② 블라인드 매칭 — 내가 먼저 쓰기 전까지 상대 일기는 잠겨있어요 (기대감!)
③ AI 월간 에세이 — 한 달치 일기를 AI가 감성 에세이로 만들고 음성으로 읽어줘요

⚠️

지금 이 앱의 상태

현재 버전은 데모(시제품)예요. 실제 서버가 없어서 내 기기에서만 데이터가 저장돼요. 파트너와 실시간 연결은 아직 안 되고, 화면 구성과 기능이 어떻게 작동하는지 체험하는 버전이에요.

02 · COVER SCREEN

첫 화면 — "구글/카카오 버튼이 왜 있어요?"

앱을 열면 가장 먼저 보이는 화면이에요. 여기서 가장 많이 헷갈리는 부분을 먼저 설명할게요.

🤔

"구글로 시작하기 / 카카오로 시작하기" 버튼이 왜 있나요?

실제 앱이라면 여기서 진짜 구글/카카오 계정으로 로그인을 해야 해요. 그래야 서버에서 내 데이터를 저장하고, 파트너와 연결하고, 다른 기기에서도 쓸 수 있거든요.

하지만 지금 이 데모 버전은 실제 서버가 없어요. 그래서 이 버튼들은 그냥 "다음 화면으로 넘어가기" 버튼처럼 작동해요. 어느 것을 눌러도 똑같이 이름 설정 화면으로 이동해요.

🔑 구글로 시작하기

구글 계정으로 로그인

실제 앱에서는 본인의 구글 계정으로 가입/로그인 해요. 데모에서는 그냥 다음 화면으로 넘어가는 버튼이에요.

👆 눌러도 됩니다. 아무 버튼이나 누르면 돼요.

💬 카카오로 시작하기

카카오 계정으로 로그인

한국 사용자는 카카오 계정이 더 편리하죠. 실제 앱에서는 카카오 로그인 창이 떠요. 데모에서는 구글 버튼과 동일하게 작동해요.

👆 이것도 눌러도 됩니다. 똑같이 다음으로 가요.

이미 계정이 있으신가요? 연결하기

초대 코드로 파트너와 연결

파트너에게 초대 코드를 받았다면 여기서 입력해서 같은 일기장을 공유할 수 있어요. 데모에서는 이름 설정 화면으로 가요.

📌

결론: 뭘 눌러야 하나요?

아무거나 눌러도 돼요! 구글이든 카카오든 데모에서는 동일하게 이름 설정 화면으로 넘어가요. 일단 눌러서 들어가세요. 또는 화면 가운데 일기장 그림을 탭해도 들어가져요.

💡

일기장 표지 그림은 뭔가요?

화면 중앙에 공중에 떠 있는 검은 책 모양이 우리 커플의 일기장 표지예요. 안에 두 사람 이름이 표시돼요. 이걸 탭하거나 왼쪽으로 스와이프해도 시작할 수 있어요.

03 · SETUP

이름 설정 화면

처음 한 번만 설정하면 돼요. 나중에 설정에서 바꿀 수 있어요.

1

내 이름 입력

일기 카드에 표시될 내 이름을 입력해요. 실명이 아니어도 돼요. 닉네임이나 애칭도 OK예요.

예) 지민, 엄마, 준혁아빠, 달링 — 10자 이내
2

파트너 이름 입력

상대방의 카드에 표시될 이름이에요. 상대가 직접 설정하는 게 아니라 내가 부르는 이름을 입력해요.

예) 서준, 여보, 딸, 우리엄마
3

초대 코드 (선택사항)

파트너에게 받은 6자리 초대 코드가 있으면 여기서 입력해요. 같은 일기장을 공유하게 돼요.
데모에서는 실제 연결이 안 되지만, 버튼을 눌러보면 "연결됐어요" 메시지가 나와요.

4

일기 시작하기 → 버튼

이 버튼을 누르면 설정이 저장되고 메인 타임라인으로 이동해요. 이름을 비워두면 자동으로 기본값(지민/서준)이 들어가요.

04 · MAIN TIMELINE

메인 화면 — 타임라인

앱의 핵심 화면이에요. 나와 파트너의 일기 카드가 나란히 쌓여있어요.

📋

화면 구조 한눈에

상단 — "커플 이름 + 이번 달 + 연속 일수"
중간 — 날짜별로 두 사람의 폴라로이드 카드가 2열로 나열
하단 — 내비게이션 바 (🏠홈 / 📷쓰기 / ✨AI결산)

📷 카드

폴라로이드 일기 카드

각 카드를 탭하면 사진이 크게 보이고, 파트너 카드에는 스티커를 붙일 수 있어요. 왼쪽 카드가 나, 오른쪽이 파트너예요.

🔒 잠금 카드

블라인드 잠금 상태

파트너가 일기를 썼는데 내가 아직 안 썼으면 파트너 카드가 흐릿하게 잠겨요. 내가 오늘 일기를 쓰는 순간 파트너 카드가 공개돼요!

💡 이게 바로 블라인드 매칭 — 기대감이 생기는 이유예요

📷

중앙 카메라 버튼 (FAB)

하단 가운데 돌출된 검은 원 버튼이에요. 누르면 오늘 일기 쓰기 화면으로 바로 이동해요. 오늘 이미 일기를 썼으면 ✓ 표시로 바뀌어요.

👆 이 버튼이 가장 자주 쓰는 버튼이에요!

🏠 홈

홈 (타임라인)

지금 보고 있는 메인 화면이에요. 다른 화면에 있다가 돌아올 때 눌러요.

✨ AI결산

AI 월간 결산

이 달의 감정 통계, AI가 쓴 에세이, 나래이션 생성 화면으로 이동해요.

⚙️

설정 버튼

상단 오른쪽 ⚙️를 탭하면 설정 화면으로 이동해요. 이름 확인, 블라인드 모드 설정, 초대 코드 복사 등을 할 수 있어요.

💡 타임라인에서 샘플 데이터가 보이는 이유

  • 앱을 처음 열면 아무것도 없어서 허전하니까, 미리 넣어둔 예시 일기 5개가 보여요.
  • 내가 일기를 쓰면 맨 위에 새 카드가 추가돼요.
  • 샘플 데이터가 싫으면 설정 → "처음부터 다시 시작"으로 초기화할 수 있어요.
05 · WRITE DIARY

일기 쓰기 — 3단계

하단 📷 버튼을 누르면 시작돼요. 딱 3단계, 1~2분이면 끝나요.

1

사진 또는 배경 선택

오늘을 대표하는 사진 1장을 올리거나, 8가지 감성 배경 중 하나를 골라요.
사진이 없어도 괜찮아요. 배경만으로도 충분히 예뻐요.

📸 점선 박스 탭 → 앨범에서 사진 선택
🎨 아래 동그란 버튼들 → 감성 배경 선택 (노을빛/달빛/초록빛 등)
2

오늘의 감정 선택

8가지 감정 아이콘 중 오늘 기분에 맞는 걸 탭해요. 하나만 고를 수 있어요.

😊 행복 🥰 설렘 😌 평온 😴 피곤 😢 슬픔 😤 답답 😆 신남 🤔 고민
감정을 고르면 "다음 → 글귀 입력" 버튼이 활성화돼요
3

한 줄 글귀 입력 (60자 이내)

오늘 하루를 한 줄로 남겨요. 길게 쓸 필요 없어요. 짧고 솔직한 문장이 더 좋아요.

예) "오늘 드디어 그 카페 갔어!" / "퇴근길 하늘이 너무 예뻤어" / "그냥 네가 보고 싶었어"
✓

일기 저장하기 버튼

빨간 버튼을 누르면 일기가 저장되고 타임라인에 폴라로이드 카드가 생겨요. 파트너의 오늘 일기가 있었다면 이 순간 공개돼요!

⚠️ 하루에 한 번만 쓸 수 있어요. 저장 후 수정은 안 돼요.
👁

오른쪽 위에 보이는 작은 폴라로이드는 뭔가요?

3단계(글귀 입력) 화면에 나오는 미리보기 카드예요. 내가 입력하는 글귀가 실시간으로 반영돼서, 완성된 카드가 어떻게 보일지 미리 확인할 수 있어요.

06 · AI REPORT

AI 결산 화면

하단 ✨ 버튼을 누르면 이동해요. 이달의 감정 흐름과 AI 에세이를 볼 수 있어요.

📊
감정 흐름 차트
이달의 감정 분포

이달에 내가 가장 많이 느낀 감정이 막대 그래프로 표시돼요.

일기를 많이 쓸수록 더 정확한 통계가 나와요.

일기가 없으면 "아직 이달의 일기가 없어요" 메시지가 나와요.

🖼
이달의 순간들
사진 스트립

이달에 올린 사진/배경들이 작게 가로로 나열돼요.

가로로 스크롤해서 더 볼 수 있어요.

✍️
AI 에세이
감성 회고 글

이달 일기의 감정·글귀를 분석해서 AI가 자동으로 쓴 감성 에세이예요.

"우리의 N월은 ..." 으로 시작하는 2~3문단 글이에요.

일기를 더 많이 쓸수록 에세이가 풍부해져요.

🎙️
나래이션
AI 음성 생성

ElevenLabs API 키를 입력하면 에세이를 실제 음성으로 들을 수 있어요.

자세한 사용법은 아래 나래이션 섹션을 봐주세요.

📤 공유하기

공유하기 버튼

에세이를 카카오톡, 인스타그램 등으로 공유해요. 기기에서 공유 기능을 지원하면 공유 창이 떠요. 지원 안 되면 "공유 링크가 복사됐어요" 메시지가 나와요.

💾 소장하기

소장하기 버튼

에세이를 저장해요. 현재 데모에서는 "저장됐어요 (데모)" 메시지만 나와요. 실제 앱에서는 PDF로 저장되거나 기기에 이미지로 저장되는 기능이에요.

07 · NARRATION

나래이션 생성 사용법

AI가 쓴 에세이를 실제 사람 목소리처럼 읽어주는 기능이에요. ElevenLabs API 키가 필요해요.

🔐

먼저 이것부터 — API 키 보안 주의

API 키는 절대 다른 사람에게 보여주거나 채팅창에 붙여넣으면 안 돼요. 다른 사람이 내 키를 쓰면 내 계정의 무료 크레딧이 소진돼요. 항상 앱 안의 입력란에만 넣어주세요.

1

ElevenLabs 회원가입 (무료)

elevenlabs.io 에 접속해서 무료로 가입해요. 신용카드 불필요, 이메일만 있으면 돼요.

2

API 키 발급

로그인 후 → 우측 상단 프로필 사진 클릭 → Profile + API Key → API Key 섹션에서 키를 복사해요. sk_ 로 시작하는 긴 문자열이에요.

3

앱의 나래이션 입력란에 붙여넣기

✨ AI결산 화면 → 상단 "ElevenLabs API 연결" 입력란에 복사한 키를 붙여넣기 → 저장 버튼 클릭. 한 번 저장하면 다음에 다시 입력 안 해도 돼요.

키가 저장되면 "🔐 저장된 API 키로 나래이션 생성 가능해요!" 메시지가 나와요
4

🎙️ 나래이션 생성하기 버튼 클릭

버튼을 누르면 5~15초 후에 실제 AI 음성이 생성돼요. 생성이 완료되면 ▶ 버튼을 눌러서 들을 수 있어요. 진행 바를 탭해서 원하는 위치로 이동도 가능해요.

⚠️ 생성 중에는 버튼이 비활성화돼요. 기다려주세요.
💰

요금이 걱정돼요

무료 플랜으로 매달 약 10분 분량을 생성할 수 있어요. 에세이 한 편이 약 1~2분이니까 월 5~10회 생성이 무료예요. 커플 두 명이 매달 한 번씩 쓴다면 무료로 충분해요.

🎧 오류가 났을 때 확인사항

  • "API 키가 올바르지 않아요" → 키를 다시 복사해서 저장해보세요
  • "무료 플랜 한도 초과" → ElevenLabs에서 이달 크레딧이 소진된 거예요. 다음 달에 리셋돼요
  • 아무 반응 없음 → 인터넷 연결을 확인해주세요

파트너가 아직 오늘 일기를 안 썼거나, 샘플 데이터여서 오늘 날짜가 아닌 경우예요. 파트너가 오늘 일기를 써야 양쪽이 공개돼요. 데모에서는 파트너가 없으니 오늘 날짜의 파트너 카드는 "⏳ 아직 작성 안 했어요"로 보여요.
앱을 껐다 켜니까 일기가 사라졌어요▼
데이터는 브라우저의 로컬 저장소(localStorage)에 저장돼요. 브라우저 데이터를 지우거나 시크릿 모드로 접속하면 데이터가 사라져요. 일반 모드로 접속하면 데이터가 유지돼요.
오늘 일기를 잘못 썼는데 수정할 수 있나요?▼
현재 데모 버전에서는 수정 기능이 없어요. 처음부터 다시 시작(설정 → 초기화)하면 전체가 삭제돼요. 실제 앱에서는 24시간 이내 수정 기능을 추가할 예정이에요.
AI 결산 화면에 감정 차트가 안 나와요▼
이번 달(현재 월)에 작성한 일기가 없으면 차트가 비어요. 샘플 데이터는 5월 날짜라서, 현재 달이 5월이 아니면 차트에 아무것도 안 나올 수 있어요. 직접 일기를 한 개라도 써보면 바로 나타나요.
파트너 일기에 스티커를 붙이려면 어떻게 해요?▼
타임라인에서 파트너 카드를 탭해서 크게 열어요. 하단 "스티커 반응 붙이기" 패널에서 ❤️ 🤗 😄 🌟 🥺 중 하나를 탭하면 사진 위에 랜덤 위치에 스티커가 붙어요. 내 카드에는 스티커 패널이 안 나와요.
나래이션 버튼을 눌렀는데 아무 반응이 없어요▼
API 키가 저장되지 않은 경우예요. AI결산 화면 상단 입력란에 ElevenLabs API 키를 넣고 저장 버튼을 눌러야 해요. 또는 API 키가 만료됐거나 잘못됐을 수 있어요. ElevenLabs에서 새 키를 발급받아보세요.
← 뒤로가기 버튼이 어디 있어요?▼
일기 쓰기, 카드 상세, AI결산, 설정 화면에서 왼쪽 상단에 동그란 ← 버튼이 있어요. 또는 하단 홈(🏠) 버튼을 눌러도 메인 화면으로 돌아와요.
앱이 폰처럼 작은 화면으로 보여요. 왜 그런가요?▼
이 앱은 모바일 앱처럼 보이도록 의도적으로 설계됐어요. PC에서 열면 화면 중앙에 폰 모양으로 나타나고, 실제 스마트폰에서 열면 화면 전체를 채워요. 모바일에서 열어보면 훨씬 자연스러워요.
