① 데이터 저장소 연결
메뉴, 후기 등 모든 내용이 저장될 공간입니다.
jsonbin.io 에서 무료로 만들 수 있어요.
1
jsonbin.io 접속 → Sign Up 으로 무료 가입
2
로그인 후 우측 상단 아이콘 → API Keys 클릭
3
Secret Key 옆 복사 버튼 클릭
올바른 Secret Key를 입력해주세요
② 저장 공간(Bin) 만들기
내 데이터만 담을 개인 공간을 만들고 ID를 붙여넣어 주세요.
1
왼쪽 메뉴 Bins → 우측 상단 Create Bin
2
내용에 { "status": "ok" } 입력 후 Create 클릭
3
BIN ID 복사
예: https://api.jsonbin.io/v3/b/6630f1a9acd...
연결에 실패했습니다. Secret Key와 Bin ID를 확인해주세요
③ 사진 저장소 연결
메뉴 사진을 올릴 공간입니다.
imgbb.com 에서 무료로 발급받아요.
1
imgbb.com 접속 → 무료 가입 & 로그인
2
api.imgbb.com 접속
3
Get API Key 클릭 → 복사
ImgBB API Key를 입력해주세요
④ 비밀번호 설정 & 북마크
관리자 페이지 진입 비밀번호를 설정하세요.
