☁️ 클라우드 초기 설정
모든 기기에서 동기화되려면 아래 2개의 무료 서비스 키가 필요합니다.
딱 한 번만 설정하면 이후엔 자동으로 작동합니다.
STEP 1 — JSONbin.io (데이터 저장소)
1. jsonbin.io 접속 → 무료 회원가입
2. 로그인 후 상단 API Keys 클릭
3. Secret Key 복사 후 아래 붙여넣기
4. 좌측 Bins → Create a Bin 클릭
5. 내용에 {} 입력 후 저장 → URL에서 Bin ID 복사
STEP 2 — ImgBB (사진 저장소)
1. imgbb.com 접속 → 무료 회원가입
2. 로그인 후 api.imgbb.com 접속
3. Get API Key 클릭 → API 키 복사
키는 이 브라우저에만 저장됩니다 (비공개). 다른 기기에서 관리자 모드 사용 시 동일하게 입력하세요.
